# Crypto AI Tools

[![Concept](https://img.shields.io/badge/status-concept-orange)](https://github.com/crypto-ai-tools)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Discussions](https://img.shields.io/badge/discussions-welcome-green)](https://github.com/crypto-ai-tools/discussions)

### Free, Open-Source AI Tools for Cryptocurrency Traders

> *No financial advice. No token recommendations. No copy-trading.*
> Just pure information tools that help you see more, understand more, and decide for yourself.

---

## Projects

| Project | Description | Status |
|---|---|---|
| [**Soulbound Wallet**](./soulbound-wallet/) | AI personality permanently bonded to a blockchain address. Trades like you, protects like you, outlives you. MPC signing, AI firewall, immortality protocol. | Concept |
| [**Whale Shadow**](./whale-shadow/) | Real-time multi-chain on-chain intelligence radar. AI anomaly detection. See whale movements, LP drains, and CEX flows before the chart moves. | Concept |
| [**SimulatorX**](./simulator-x/) | Pre-trade simulation engine. 10,000 Monte Carlo paths. MEV threat modeling. Order book depth. Concurrent trader behavior. Know the weather before you step outside. | Concept |

---

## Philosophy

- **Free forever.** All tools are open source (MIT). No premium tiers. No hidden fees.
- **Information, not advice.** These tools surface data and probabilities. You make the decisions.
- **Radically transparent.** Every confidence score, model limitation, and assumption is documented.
- **Community-first.** Contributions welcome. Roadmap driven by user needs, not investor demands.

---
